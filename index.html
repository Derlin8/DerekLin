Uno Card Game

One of my class projects was to design a Uno Card Game. With the different colors, numbers, and actions a player could perform, this made the task of programming difficult. Using object-oriented fundamentals such as abstraction, encapsulation, and polymorphism, I truncated the problem into smaller pieces, making implementation less difficult.

Pizza Button

I used a lambda function to execute a python function via an AWS IoT button trigger. This simple command accessed the Domino's Pizza Chain restraunt, prompting them to deliver a simple pepperoni pizza using Domino's API. This project showed me the connection between programming and real-life applications. It also inspired me to explore projects related to IoT.
